# OrangeHRM QA Testing Project

## Project Overview

This repository contains the first phase of a comprehensive Quality Assurance testing project for OrangeHRM Open Source.

The goal of this project is to apply real-world QA practices and build a complete testing portfolio covering different testing approaches and deliverables used in software testing projects.

## Project Scope

The current phase focuses on Manual Testing activities including:

* Test Planning
* Test Case Design
* Test Execution
* Defect Reporting

Modules covered:

* Authentication
* Dashboard
* PIM (Employee Management)
* Admin (User Management)

## Deliverables

### Test Plan

A detailed test plan defining:

* Testing objectives
* Scope
* Testing types
* Test environment
* Risks and mitigation
* Deliverables
* Success criteria

### Test Cases

Designed and executed test cases covering:

* Positive scenarios
* Negative scenarios
* Validation testing
* Functional testing
* UI verification

### Bug Reports

Documented defects discovered during execution including:

* Forgot Password 504 Gateway Timeout
* Invalid Username Password Reset Behavior
